# 🌌 AURA AI

> **Where Ambition Meets Intelligence**

AURA AI is an AI-powered Career & Learning Platform designed to help students and aspiring professionals accelerate their learning and career growth through intelligent assistance. Built with a futuristic, space-inspired interface, AURA AI combines multiple AI tools into a single seamless experience.

---

## ✨ Features

- 💬 **Career Chat**
  - AI-powered career guidance
  - Coding and technical Q&A
  - Personalized learning support

- 📄 **Resume Studio**
  - ATS Resume Analysis
  - Resume Review
  - Skill Gap Detection
  - Resume Improvement Suggestions

- 📚 **PDF Assistant**
  - Chat with PDF documents
  - Study notes assistant
  - Research paper summarization
  - Context-aware question answering

- 🎤 **Interview Preparation**
  - AI-generated interview questions
  - Resume-based interview practice
  - Technical and HR preparation

- 🗺️ **Learning Roadmaps**
  - Personalized learning paths
  - Career-focused skill development

- 🌌 **Premium User Experience**
  - Space-inspired UI
  - Glassmorphism design
  - Responsive layout
  - Elegant dark theme

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/snehanreddy-05/AURA_AI.git
````

Move into the project directory

```bash
cd AURA_AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
