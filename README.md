# 🤖 AI Study & Career Assistant

## Overview

AI-powered assistant that helps students with learning, resume analysis, interview preparation, and PDF-based question answering.

## Features

* AI Chatbot
* PDF Upload & Summarization
* PDF Question Answering
* OCR Support for Scanned PDFs
* Resume Analyzer
* Interview Question Generator
* Skill Gap Analyzer
* ATS Resume Score

## Technologies Used

* Python
* Streamlit
* OpenRouter API
* OCR (Tesseract)
* PDF Processing
* Machine Learning Concepts

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
